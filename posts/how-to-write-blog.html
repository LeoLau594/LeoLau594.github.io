如何在这个博客中添加新文章

2025年3月18日 教程 博客

欢迎来到我的博客！这篇文章将详细介绍如何在这个基于纯 HTML/CSS/JavaScript 构建的静态博客中添加新文章。

准备工作

在开始写作之前，你需要准备以下内容：

  • 文章标题
  • 发布日期
  • 文章标签（1-3个）
  • 文章摘要（用于列表展示）
  • 文章内容

步骤一：创建文章文件

最简单的方法是复制现有的文章模板：

  1. 复制 posts/template.html 文件
  2. 重命名为你的文章文件名（如：my-new-post.html）
  3. 放在 posts/ 目录下

步骤二：编辑文章信息

打开新创建的文件，修改以下关键部分：

1. 页面标题和描述

<meta name="description" content="文章标题 - Leo的个人博客">
<title>文章标题 | Leo's Blog</title>

2. 文章标题

<h1 class="article-title">你的文章标题</h1>

3. 发布日期

2025年3月18日

4. 标签

<span class="tag">标签1</span>
<span class="tag">标签2</span>

步骤三：编写内容

在 <div class="article-content"> 中编写你的文章内容。支持以下格式：

标题

<h2 id="section-id">第二节</h2>

段落

<p>这是段落内容...</p>

代码块

<pre><code class="language-javascript">
function hello() {
  console.log('Hello World!');
}
</code></pre>

引用

<blockquote>
  这是一段引用文字
</blockquote>

步骤四：更新导航

在文章底部更新上一篇/下一篇导航链接：

<nav class="nav-arrows">
  <a href="上一篇.html" class="nav-arrow prev">...</a>
  <a href="下一篇.html" class="nav-arrow next">...</a>
</nav>

步骤五：添加到列表

最后，你需要将文章添加到列表页面：

更新 posts.html

在 <div class="blog-grid"> 内添加新的文章卡片（复制现有卡片并修改）。

更新 index.html

同样在首页的文章列表中添加新卡片，或者替换最旧的一篇以保持只显示 3 篇最新文章。

写作小贴士

选择渐变色

文章卡片支持自定义渐变色背景，以下是一些推荐配色：

  • 紫色系：#667eea → #764ba2
  • 粉色系：#f093fb → #f5576c
  • 蓝色系：#4facfe → #00f2fe
  • 绿色系：#43e97b → #38f9d7

文章摘要

摘要会显示在文章列表中，建议控制在 50-100 字，简洁概括文章要点。

💡 提示：你可以参考 posts/template.html 模板文件和 写作指南.md 获取更详细的说明。

完成！

按照以上步骤，你就可以成功发布新文章了。现在打开浏览器预览效果吧！
