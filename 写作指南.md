# 📝 博客写作指南

## 添加新文章的步骤

### 步骤 1：创建文章文件

1. 复制 `posts/template.html` 文件
2. 重命名为你的文章文件名（建议使用英文，如：`my-new-post.html`）
3. 放在 `posts/` 目录下

### 步骤 2：编辑文章内容

修改以下关键部分：

#### A. 页面头部信息（第 6-7 行）
```html
<meta name="description" content="文章标题 - Leo的个人博客">
<title>文章标题 | Leo's Blog</title>
```

#### B. 文章标题（第 77 行）
```html
<h1 class="article-title">你的文章标题</h1>
```

#### C. 发布日期（第 82 行）
```html
2025年1月1日
```

#### D. 标签（第 88-89 行）
```html
<span class="tag">技术</span>
<span class="tag">教程</span>
```

#### E. 文章内容（第 95 行开始）
```html
<div class="article-content">
  <h2 id="section1">第一节标题</h2>
  <p>这里是段落内容...</p>
  
  <h2 id="section2">第二节标题</h2>
  <p>更多内容...</p>
</div>
```

### 步骤 3：更新文章导航

在文件底部（约 107-127 行），修改上一篇/下一篇链接：

```html
<nav class="nav-arrows">
  <a href="上一篇文章的文件名.html" class="nav-arrow prev">
    ...
    <div class="nav-arrow-title">上一篇文章标题</div>
  </a>
  <a href="下一篇文章的文件名.html" class="nav-arrow next">
    ...
    <div class="nav-arrow-title">下一篇文章标题</div>
  </a>
</nav>
```

如果是最新文章，删除 "下一篇" 部分的代码。

### 步骤 4：添加到文章列表

**更新 `posts.html`**：

在 `<div class="blog-grid">` 内添加新的文章卡片（复制现有卡片修改）：

```html
<article class="card">
  <div class="card-image" style="background: linear-gradient(135deg, #颜色1 0%, #颜色2 100%);"></div>
  <div class="card-content">
    <div class="card-meta">
      <span class="card-date">
        <svg ...>日历图标</svg>
        2025年1月1日
      </span>
    </div>
    <h3 class="card-title">
      <a href="posts/你的文章.html">文章标题</a>
    </h3>
    <p class="card-excerpt">文章简介/摘要...</p>
    <div class="card-tags">
      <span class="tag">标签1</span>
      <span class="tag">标签2</span>
    </div>
  </div>
</article>
```

**更新 `index.html`**：

同样在 `<div class="blog-grid">` 内添加（或替换最旧的文章卡片以保持只显示 3 篇最新）

## 🎨 文章样式指南

### 标题层级
- `<h2>` - 主要章节
- `<h3>` - 子章节

### 代码块
```html
<pre><code class="language-javascript">
你的代码
</code></pre>
```

### 引用
```html
<blockquote>
  引用内容
</blockquote>
</blockquote>
```

### 图片
```html
<img src="图片地址" alt="图片描述">
```

### 链接
```html
<a href="链接地址" target="_blank">链接文字</a>
```

### 列表
```html
<ul>
  <li>项目1</li>
  <li>项目2</li>
</ul>
```
