关于我

在代码与项目之间，寻找属于自己的节奏

👋 你好，我是 Leo

一个在深圳搬砖的 IT 从业者，白天和系统、数据、需求打交道，晚上偶尔写写代码、折腾折腾个人项目。

过去几年一直在企业数字化这个领域摸爬滚打，从主数据管理到系统集成，从低代码平台到项目交付，踩过不少坑，也攒了一些经验。虽然工作偏项目管理，但骨子里还是喜欢技术，相信好的技术能解决很多看似复杂的业务问题。

💻 平时在捣鼓什么

项目管理 主数据管理 系统集成 低代码平台 SQL Web开发

工作相关主要是 MDM、ESB、ERP 这些传统企业软件，业余时间喜欢研究一些轻量级的技术栈，比如这个博客就是用纯 HTML/CSS/JS 写的，没有框架、没有构建工具，简单直接。

🎯 为什么写这个博客

主要有几个原因：

  • 记录 — 工作中遇到的问题和解决方案，记下来方便自己也方便他人
  • 梳理 — 写出来是检验自己是否真正理解的好方法
  • 分享 — 数字化、项目管理这些领域的中文优质内容还不够多，希望能贡献一份力量
  • 折腾 — 纯粹想自己动手做一个好看的个人网站

🌱 一些喜好

  • 喜欢简洁的设计，信奉「少即是多」
  • 偏爱命令行工具，觉得终端有一种独特的美感
  • 偶尔打游戏，主要是单机 RPG 和策略类
  • 喜欢下雨天，觉得安静适合思考

📮 想聊聊？

欢迎通过邮件或 GitHub 与我交流，聊技术、聊项目、或者单纯吐槽工作都可以。看到会尽快回复。

📝 关于这个站

这是一个静态博客，托管在 GitHub Pages 上。设计灵感来源于赛博朋克风格，深色模式下有霓虹灯效果和扫描线动画。博客主题支持一键切换深色/浅色模式，响应式布局适配各种设备。

本站不使用 Cookies 收集个人信息，不加载第三方追踪脚本，尽可能保持轻量和隐私友好。
